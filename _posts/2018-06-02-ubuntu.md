## 编译最新CMake
```
wget https://cmake.org/files/v3.11/cmake-3.11.0.tar.gz

tar -xzf cmake-3.11.0.tar.gz
cd cmake-3.11.0/
```

## 创建快捷方式
```
sudo apt-get install --no-install-recommends gnome-panel
gnome-desktop-item-edit --create-new ~/桌面
```
